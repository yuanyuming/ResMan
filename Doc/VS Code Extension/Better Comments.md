# Better Comments
Create human-friendly comment.

## Annotation Categorise
- Alerts
- Queries
- TODOs
- Highlights
- Commented Out Code
- Other

## Configuration

### MultilineComments Styled
`"better-comments.multilineComments":true`

### Highlight Plain Text
Styled plain text file 
`"better-comments.highlightPlainText": false`
